Добро пожаловать к softу для домашнего хранилища

Варианты которые можно рассмотреть

Xigmanas

OpenSource проект, можно установить на usb флешку

TrueNas

Платформа для хранения данных на файловой системе Zfs. Может применятся в корпорациях и дома. Требует: 8 ГБ ОЗУ, 2 ядра CPU

OpenmediaVault

Os на Debian Linux. Требования 256 mb, 2 gb диска. Поддержка контейнеров

Easynas

OS для nas на основе openSuse, fs Btrfs можно расширить хранилище без остановки, слабая безопасность, только для дома
